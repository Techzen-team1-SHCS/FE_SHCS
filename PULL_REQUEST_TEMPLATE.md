## Loại thay đổi (Type of change)
Vui lòng đánh dấu vào các ô phù hợp:
- [ ] 🐛 Bug fix (Non-breaking change sửa lỗi hệ thống)
- [ ] ✨ New feature (Non-breaking change thêm mới chức năng)
- [ ] 💥 Breaking change (Thay đổi cấu trúc/API có thể ảnh hưởng đến module khác)
- [ ] ♻️ Refactor (Code cải thiện cấu trúc, không thay đổi logic hiện tại)
- [ ] 📝 Documentation update (Cập nhật tài liệu)
- [ ] 🔧 Chore (Cập nhật configurations, packages...)

## Checklist (Cần kiểm tra trước khi Approve)
Vui lòng tự review mã nguồn trước khi yêu cầu review từ Lead:
- [ ] Tính năng hoạt động ổn định trên môi trường Local.
- [ ] Code đã được kiểm tra chất lượng bằng `npm run lint` và format lại bằng `npm run format`.
- [ ] Không có conflict với nhánh gốc (đã pull & merge nhánh `dev`/`main` mới nhất).
- [ ] Code không có cảnh báo warning, error trong Console.
- [ ] Đã đính kèm ảnh chụp (Screenshot)/Video demo minh chứng vào Ticket hoặc PR này.
- [ ] Các tên biến/hàm tuân thủ chuẩn Coding Rules.
