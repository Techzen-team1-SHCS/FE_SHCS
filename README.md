# Hotel Booking - Frontend (ReactJS)

## 🚀 Cài đặt
### Yêu cầu
- Node.js >= 18
- npm hoặc yarn

### Các bước
```bash
# Di chuyển vào thư mục dự án
cd hotelbookingfe

# Cài đặt dependencies
npm install   # hoặc yarn install

# Chạy ứng dụng
npm run dev
## 🛠 Tools và Thư viện
- **React 18.2.0**: Framework chính
- **Vite 4.4.5**: Build tool
- **React Router DOM 7.9.1**: Routing
- **TanStack React Query 5.90.12**: Data fetching
- **Axios 1.13.1**: HTTP client
- **Styled Components 6.1.19**: CSS-in-JS
- **Formik 2.4.6**: Form management
- **Yup 1.7.1**: Validation
- **React OAuth Google 0.12.2**: Google auth
- **Pusher JS 8.4.0**: Real-time messaging
- **Laravel Echo 2.2.4**: Broadcasting
- **ESLint 8.45.0**: Code linting
- Và các thư viện UI khác: Swiper, React Icons, React Toastify, SweetAlert2, etc.'
